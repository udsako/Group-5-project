## Tech Stack

- **Frontend:** React (in progress)
- **Database:** TBD

## Getting Started

### Backend
cd backend
npm install
npm run dev

## API Endpoints
| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | /api/events | Get all events |
| GET | /api/events/:id | Get single event |
| GET | /api/bookings | Get all bookings |
| POST | /api/bookings | Create a booking |
